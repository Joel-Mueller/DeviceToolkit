# Homebrew Installation

Download Homebrew from the site -> https://brew.sh

## Cask

```shell
xargs -I {} brew install --cask {} < apps.txt
```

```txt
visual-studio-code
spotify
discord
brave-browser
github
rectangle
zoom
docker
anki
```

## Formulas

```shell
xargs -I {} brew install {} < brew.txt
```
